# Zomato Dataset EDA

This repository contains my Exploratory Data Analysis (EDA) on the **Zomato Restaurant Dataset** done in a Kaggle Notebook.

## Objectives
- Understand the structure of the dataset  
- Clean data and handle missing values  
- Analyze restaurant ratings, price range, and location-wise trends  
- Visualize important patterns using graphs

## Tools & Libraries
- Python  
- pandas, numpy  
- matplotlib, seaborn  

## Notebook
The main analysis is in the Jupyter notebook file in this repository.
